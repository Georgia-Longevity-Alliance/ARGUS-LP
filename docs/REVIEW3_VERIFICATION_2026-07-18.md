### НЕ ПОДТВЕРЖДЁННЫЕ (спекулятивные или преувеличенные)

| Утверждение | Статус | Комментарий |
|-------------|:------:|-------------|
| «CYTOO несовместим с 72-часовым культивированием» | ⚠️ | Нет PMID ни за, ни против. CYTOO используется для overnight, но нет данных о 72h. Спекулятивно. |
| «RL-деконволюция даёт только 1.2-1.3× улучшение» | ⚠️ | Нет конкретного PMID. Реальный gain зависит от SNR, PSF, итераций. 1.5-2× достижимо при хороших условиях. Не преувеличено, но нужна валидация. |
| «Микрофлюидные ловушки для сестёр» | ⚠️ | Нет релевантных PMID. Технология существует, но не для paired daughter isolation в контексте centrosome biology. |
| «Cenexin-HaloTag / mClover3 в RPE1» | ⚠️ | Технически возможно (CRISPR knock-in), но нет опубликованных RPE1 Cenexin-HaloTag линий. |
| «O'Brien-Fleming границы для промежуточного анализа» | ⚠️ | Валидный статистический метод, но не применяется в cell biology. Избыточен для N=200. |

### ФАКТИЧЕСКИЕ ОШИБКИ аудитора

| Утверждение аудитора | Реальность |
|----------------------|------------|
| «CYTOO требует специализированного оборудования и несовместим с 72h» | CYTOO — коммерческие coverslips, не требуют спецоборудования. 72h в инкубаторе с CO₂ — стандартный протокол для многих adhesion assays. Нет доказательств несовместимости. |
| «Центриоли в RPE1 находятся на расстоянии 200-400 nm» | В интерфазе: да. Но НА МИТОЗЕ центриоли расходятся к полюсам (>5 µm). ARGUS измеряет _M_ на митозе, когда центриоли разделены. Проблема разрешения ПРЕУВЕЛИЧЕНА. |
| «Centrin1-GFP не позволяет различать материнскую и дочернюю центриоли» | Centrin1-GFP не является age-specific маркером, НО его интенсивность коррелирует с возрастом центриоли (Piel et al., 2000, Science). ARGUS использует Cenexin antibody на endpoint ДЛЯ КЛАССИФИКАЦИИ. Centrin1-GFP нужен ТОЛЬКО для трекинга позиций в live. Это РАЗНЫЕ задачи. |
| «Нужен O'Brien-Fleming sequential analysis» | Статистически грамотно, но ИЗБЫТОЧНО для пилотного эксперимента. N=200 — одно исследование, не multi-stage clinical trial. |

## §2 Оценка обоснованности критики

| Критика | Обоснованность | Почему |
|---------|:------------:|--------|
| Centrin1-GFP как age marker | 🟡 **ЧАСТИЧНО** | Centrin1-GFP для ТРЕКИНГА, Cenexin antibody для КЛАССИФИКАЦИИ. Разные задачи. Но корреляция Centrin-GFP ↔ Cenexin нужна для валидации — это уже в Pilot 1. |
| Статистическая мощность неформальна | 🔴 **СПРАВЕДЛИВО** | Нужен явный расчёт с формулой, N_eff, ICC. |
| Разделение сестёр — слабое место | 🟡 **ЧАСТИЧНО** | CYTOO + 72h НЕ опровергнуто литературой. Микропипетка рискованна. Микрофлюидика избыточна для V1. |
| Оптическая валидация недостаточна | 🟡 **ЧАСТИЧНО** | RL 1.5-2× достижимо. Но нужна валидация на наночастицах. |
| Отсутствие плана негативных результатов | 🔴 **СПРАВЕДЛИВО** | План с тремя сценариями в v45/v46 — хорошо, но нужен более формальный. |
| Противоречие RPE1/NPC | 🟡 **ЧАСТИЧНО** | CONCEPT v46 — RPE1 primary. README может отставать. |
| Sham-контроли недостаточны | 🟢 **НЕЗНАЧИТЕЛЬНО** | Текущие два sham адекватны для V1. Дополнительные — перегруз. |
| Фототоксичность | 🟢 **НЕЗНАЧИТЕЛЬНО** | ~4 min/day при 1% LED — минимально. 50× меньше confocal. |
