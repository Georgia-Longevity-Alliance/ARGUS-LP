## Отличия от основной платформы (CONCEPT.md v93 / STATE.md v50)

| Параметр | Платформа v1.0 (CONCEPT) | V6 Прототип (Alex) | Причина |
|----------|:---:|:---:|---------|
| **Объектив** | 60×/1.2 NA WI | 40×/0.75 сухой | Дешевле в разы. Для прототипа — достаточно. |
| **Схема** | Инвертированная (стандарт OF) | Инвертированная | OpenFlexure заточен под неё. Прямая → V7. |
| **Клетки** | RPE1-hTERT | U2OS (или HeLa) | Дешевле, быстрее делятся (~22 ч), неприхотливые. Доставка отработана. |
| **ИИ-агент** | Jetson Orin NX + Hailo-8L (локально) | Облачный инференс | На старте — облако ($ десятки). Локальный → V7. |
| **Моторы** | TMC2209 + NEMA-17 | Штатные 28BYJ-48 | OpenFlexure стандарт. NEMA-17 избыточны для MVP. |
| **CO₂ датчик** | Sensirion SCD30 | SprintIR-6S | Поправка инженера Алексея. |
| **Маркеры** | Centrin1-GFP + Cenexin | Опционально. Brightfield для MVP. | Если успеваем — Centrin1-GFP. Нет — ловим митоз в brightfield. |
| **Бюджет** | $59K (v1.0) | $3-4K | Только MVP: механика + оптика + RasPi + культура. |
| **Бокс** | Инкубатор Binder + glove-box ($12.5K) | Минимальный. В инкубаторе или без. | Инкубатор — если есть доступ. Иначе — комнатная t° + частый мониторинг. |
