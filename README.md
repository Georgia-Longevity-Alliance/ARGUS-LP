# README — ARGUS-OS1

ARGUS-OS1: открытая платформа для отслеживания центриолей и записи траекторий делений.

**3 маркера:** Centrin1-GFP + SiR-Tubulin + Arl13B.
**Траектория:** бинарная (∥ или ⟂ — вдоль или поперёк предыдущего деления).
**Возраст:** количество пройденных делений.
**Судьба:** наличие цилии (Arl13B⁺).

🔗 OS1 → OS2 (lineage-память) → OS3 (прогениторные карты).

**Бюджет:** ~$23,000 | **N:** 600 пар | **Срок:** 20 недель
